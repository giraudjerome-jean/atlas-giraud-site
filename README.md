# Atlas Giraud

## Installation

```bash
npm install
npm run dev
```

Puis ouvre l’URL affichée par Vite.

## Où mettre les covers

Mets toutes les images JPG dans :

```txt
public/covers/
```

Les noms doivent correspondre à `cover_file` dans `src/data/projects_clean.json`.

## Données

Le fichier projets est ici :

```txt
src/data/projects_clean.json
```
